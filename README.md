# DeMar Logistics

Internal brokerage operations app for DeMar Transportation.

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- Supabase

## Local development

```sh
npm install
npm run dev
```

For the live broker-to-driver verification flow, read [docs/e2e-live-dispatch.md](./docs/e2e-live-dispatch.md) first.

## Build

```sh
npm run build
```

## Deployment

Deploy from the `main` branch connected to the production site at `https://logistics.demartransportation.com`.
